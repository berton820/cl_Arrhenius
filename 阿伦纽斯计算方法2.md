这是一份为你量身定制的、从零开始推导计算的TC4钛合金分段式应变补偿本构模型教学文档。
对于应用于航空航天螺栓或核电级高强度紧固件的精密锻造件而言，由于其服役环境对可靠性要求极高，基础的全局本构模型往往无法精确描述材料在复杂热力耦合下的微观组织演变。TC4在 882°C 附近发生的 $\alpha+\beta \rightarrow \beta$ 相变，会导致热变形激活能 $Q$ 发生显著跃迁。因此，建立包含相变点分段与自适应异常点剔除的本构模型，是实现高精度流变应力预测的必由之路。
本教程将严密遵循数学与物理逻辑，为你提供无死角的推导过程，可直接作为学术论文或基金申请书的理论支撑，并完全贴合 Python 自动化编程的架构。
核心物理方程与目标
我们的终极目标是建立流变应力 $\sigma$ 与应变速率 $\dot{\varepsilon}$、绝对温度 $T$ 以及真应变 $\varepsilon$ 的全映射函数。
描述这种关系的基础是包含 Zener-Hollomon 参数（$Z$ 参数）的 Sellars-Tegart 双曲正弦方程：

$$\dot{\varepsilon} = A [\sinh(\alpha \sigma)]^n \exp\left(-\frac{Q}{RT}\right)$$
其中 $R$ 为气体常数（8.314 J/(mol·K)）。我们需要通过一套闭环算法，求解出四个材料常数（$\alpha, n, Q, \ln A$），并将它们转化为关于真应变 $\varepsilon$ 的多项式函数，且在 882°C 上下采用两套独立的参数体系。
第一阶段：数据预处理与物理分流 (Data Stratification)
在这一阶段，我们将原始的连续流变曲线离散化，并根据材料的相变物理特性进行数据集隔离。
1. 物理温区分流
TC4 钛合金的相变点约为 882°C。
输入：包含 Temperature, StrainRate, TrueStrain, FlowStress 四列的 Excel 数据。将摄氏度转化为开尔文温度：$T = \text{Temperature} + 273.15$。
分流法则：
低温区（$\alpha+\beta$ 相区）：提取所有 $T < 882^\circ\text{C}$ 的实验数据，建立独立数据池 A。
高温区（$\beta$ 相区）：提取所有 $T \ge 882^\circ\text{C}$ 的实验数据，建立独立数据池 B。
注意：接下来的所有计算（第二阶段和第三阶段），数据池 A 和 B 必须完全独立运行，互不干扰。
2. 等应变切片 (Strain Slicing)
设定离散应变点：设定你需要研究的真应变范围，例如从 0.10 到 0.70，步长为 0.05。形成目标应变序列：$\varepsilon_{target} = [0.10, 0.15, 0.20, \dots, 0.70]$。
插值提取：对于每一组固定的温度和应变速率组合（即单条流变曲线），利用三次样条插值算法，提取出在每个 $\varepsilon_{target}$ 下对应的流变应力 $\sigma$。
第二阶段：带有异常剔除的逐点求解算法
针对上述切分出的每一个独立应变点（例如低温区 $\varepsilon = 0.3$），我们需要执行以下完整的数学推导与计算。这里引入基于拟合优度（$R^2$）的自适应清洗机制。
核心引擎：自适应异常点剔除线性回归
在下面的每一步求解中，我们都需要求解一元线性回归 $Y = kX + b$。为了避免相变过渡区或实验波动带来的数据失真，必须执行以下算法闭环：
对给定的 $X$ 和 $Y$ 数组进行最小二乘法拟合，计算相关系数的平方 $R^2$。
设定判定阈值（通常 $R^2 \ge 0.95$ 为有效）。
如果 $R^2 < 0.95$，计算每个点到拟合直线的残差 $\Delta_i = |Y_i - (kX_i + b)|$。
定位残差最大的异常点，将其从当前数组中剔除。
使用剩余数据重新拟合，直到 $R^2 \ge 0.95$ 或剩余数据点数量达到安全底线（如 3 个点）。提取最终的有效斜率 $k$。
步骤 2.1：求解应力乘子 $\alpha$
$\alpha$ 的物理意义是调节应力水平，公式为 $\alpha = \beta / n_1$。
求解 $n_1$（低应力近似）：
$$\dot{\varepsilon} = A_1 \sigma^{n_1} \exp\left(-\frac{Q}{RT}\right)$$
两边取对数并假定温度不变：
$$n_1 = \left( \frac{\partial \ln\dot{\varepsilon}}{\partial \ln\sigma} \right)_T$$
操作：在当前应变下，以 $\ln\sigma$ 为 $X$，$\ln\dot{\varepsilon}$ 为 $Y$。对每个恒温组分别调用“自适应回归算法”，提取各组的有效斜率并求平均值，得到 $n_1$。
求解 $\beta$（高应力近似）：
$$\dot{\varepsilon} = A_2 \exp(\beta \sigma) \exp\left(-\frac{Q}{RT}\right)$$
两边取对数并假定温度不变：
$$\beta = \left( \frac{\partial \ln\dot{\varepsilon}}{\partial \sigma} \right)_T$$
操作：以 $\sigma$ 为 $X$，$\ln\dot{\varepsilon}$ 为 $Y$。各温度组调用算法，提取斜率求平均，得到 $\beta$。
计算：得到当前应变下的 $\alpha = \beta / n_1$。
步骤 2.2：求解主应力指数 $n$
将求出的 $\alpha$ 代入主方程的对数形式：

$$\ln\dot{\varepsilon} = \ln A + n \ln[\sinh(\alpha \sigma)] - \frac{Q}{RT}$$
假定温度不变：

$$n = \left( \frac{\partial \ln\dot{\varepsilon}}{\partial \ln[\sinh(\alpha \sigma)]} \right)_T$$
操作：以 $\ln[\sinh(\alpha \sigma)]$ 为 $X$，$\ln\dot{\varepsilon}$ 为 $Y$。各温度组调用回归算法，提取有效斜率求平均，得到当前应变下的稳态应力指数 $n$。
步骤 2.3：求解热变形激活能 $Q$
对主方程分离变量，假定应变速率不变：

$$Q = R \cdot n \cdot \left( \frac{\partial \ln[\sinh(\alpha \sigma)]}{\partial (1/T)} \right)_{\dot{\varepsilon}}$$
操作：以 $1/T$ 为 $X$，$\ln[\sinh(\alpha \sigma)]$ 为 $Y$。按恒定应变速率分组调用回归算法。这里的异常剔除极为关键，它会自动剥离偏离 Arrhenius 线性规律的相变游离点。提取各速率组的斜率平均值 $S$，计算 $Q = R \times n \times S$。
步骤 2.4：求解结构因子常数 $\ln A$
引入温度补偿参数 $Z$：

$$Z = \dot{\varepsilon} \exp\left(\frac{Q}{RT}\right)$$
将主方程取对数：

$$\ln Z = \ln A + n \ln[\sinh(\alpha \sigma)]$$
操作：计算出当前应变下所有有效数据点的 $\ln Z$。以 $\ln[\sinh(\alpha \sigma)]$ 为 $X$，$\ln Z$ 为 $Y$，将当前温区（低温或高温）的所有点放在一起进行全局回归（调用异常剔除算法）。该直线的截距即为 $\ln A$。
第三阶段：多项式拟合（连续化处理）
完成第二阶段的循环后，对于你设定的目标应变序列（例如 13 个点），你将在低温区和高温区各得到 13 组独立的 $(\alpha, n, Q, \ln A)$。
为了建立参数与真应变的数学映射，针对两套数据池，分别进行 5 次多项式拟合：
低温区多项式矩阵 ($T < 882^\circ\text{C}$)：

$$\alpha_{low}(\varepsilon) = B_0 + B_1\varepsilon + B_2\varepsilon^2 + B_3\varepsilon^3 + B_4\varepsilon^4 + B_5\varepsilon^5$$

$$n_{low}(\varepsilon) = C_0 + C_1\varepsilon + C_2\varepsilon^2 + C_3\varepsilon^3 + C_4\varepsilon^4 + C_5\varepsilon^5$$

$$Q_{low}(\varepsilon) = D_0 + D_1\varepsilon + D_2\varepsilon^2 + D_3\varepsilon^3 + D_4\varepsilon^4 + D_5\varepsilon^5$$

$$\ln A_{low}(\varepsilon) = E_0 + E_1\varepsilon + E_2\varepsilon^2 + E_3\varepsilon^3 + E_4\varepsilon^4 + E_5\varepsilon^5$$
高温区多项式矩阵 ($T \ge 882^\circ\text{C}$)：
同理，拟合出 $\alpha_{high}(\varepsilon)$, $n_{high}(\varepsilon)$, $Q_{high}(\varepsilon)$, $\ln A_{high}(\varepsilon)$ 四个多项式方程。
(注：在 Python 中使用 numpy.polyfit 即可获取上述共 48 个系数，这些系数构成了材料的核心数字孪生基石)。
第四阶段：模型重构与高精度流变应力预测
在已知任意变形条件（$T, \dot{\varepsilon}, \varepsilon$）下，我们需要将数学模型反解，以预测理论流变应力 $\sigma_{pred}$。
由 $Z = A(\varepsilon) [\sinh(\alpha(\varepsilon) \sigma)]^{n(\varepsilon)}$，利用反双曲正弦函数的对数表达式 $\text{arsinh}(x) = \ln(x + \sqrt{x^2 + 1})$ 进行代数反推，结合物理分区的边界条件，得出最终的分段预测闭环方程：

$$\sigma_{pred} = \begin{cases} \frac{1}{\alpha_{low}(\varepsilon)} \ln \left\{ \left(\frac{Z_{low}}{A_{low}(\varepsilon)}\right)^{\frac{1}{n_{low}(\varepsilon)}} + \sqrt{ \left(\frac{Z_{low}}{A_{low}(\varepsilon)}\right)^{\frac{2}{n_{low}(\varepsilon)}} + 1 } \right\}, & \text{for } T < 882^\circ\text{C} \\ \frac{1}{\alpha_{high}(\varepsilon)} \ln \left\{ \left(\frac{Z_{high}}{A_{high}(\varepsilon)}\right)^{\frac{1}{n_{high}(\varepsilon)}} + \sqrt{ \left(\frac{Z_{high}}{A_{high}(\varepsilon)}\right)^{\frac{2}{n_{high}(\varepsilon)}} + 1 } \right\}, & \text{for } T \ge 882^\circ\text{C} \end{cases}$$
其中，不同温区下的 $Z$ 参数计算需要调用各自独立的激活能函数：
$Z_{low} = \dot{\varepsilon} \exp\left[ \frac{Q_{low}(\varepsilon)}{RT} \right]$
$Z_{high} = \dot{\varepsilon} \exp\left[ \frac{Q_{high}(\varepsilon)}{RT} \right]$
最后，将预测出的 $\sigma_{pred}$ 数组与实验原始应力 $\sigma_{exp}$ 进行对比，计算分段的平均绝对相对误差（AARE）。凭借引入了剔除机制和相变分段，该模型的 AARE 通常可压低至 5% 以下。
