
第一章：基础本构方程的数学解构
我们的核心目标是建立流变应力 $\sigma$、应变速率 $\dot{\varepsilon}$、变形温度 $T$ 以及真应变 $\varepsilon$ 之间的全映射关系。
对于任意一个固定的真应变状态（例如 $\varepsilon = 0.3$），包含 Zener-Hollomon 参数的 Sellars-Tegart 双曲正弦模型如下：

$$\dot{\varepsilon} = A [\sinh(\alpha \sigma)]^n \exp\left(-\frac{Q}{RT}\right)$$
为了求解未知参数（$A, \alpha, n, Q$），我们需要对该高度非线性的方程进行对数线性化处理，将其转化为多元一次方程。对方程两边同时取自然对数：

$$\ln\dot{\varepsilon} = \ln A + n \ln[\sinh(\alpha \sigma)] - \frac{Q}{RT}$$
这是一个包含四个未知数的核心基础方程。接下来的所有推导，本质上都是在控制变量（偏微分）的条件下，通过线性回归求解常数。
第二章：参数离散化与逐点求解（单应变点的计算）
在实际操作中，我们需要将连续的真应变切分成离散的点（如 $\varepsilon = 0.1, 0.15, 0.2 ...$）。对于切分出来的每一个应变点，必须完整执行以下推导与计算步骤。
第一步：求解应力乘子 $\alpha$
$\alpha$ 的物理意义是调节应力水平，它无法直接求出，需要借助低应力和高应力下的近似方程过渡。
1. 低应力近似（幂函数）
当 $\alpha\sigma < 0.8$ 时，$\sinh(\alpha\sigma) \approx \alpha\sigma$，主方程退化为：

$$\dot{\varepsilon} = A_1 \sigma^{n_1} \exp\left(-\frac{Q}{RT}\right)$$
两边取对数并假定温度 $T$ 不变，对 $\ln\sigma$ 求偏导：

$$n_1 = \left( \frac{\partial \ln\dot{\varepsilon}}{\partial \ln\sigma} \right)_T$$
操作：在当前应变下，以 $\ln\sigma$ 为横坐标，$\ln\dot{\varepsilon}$ 为纵坐标作图，各温度下拟合直线的平均斜率即为 $n_1$。
2. 高应力近似（指数函数）
当 $\alpha\sigma > 1.2$ 时，$\sinh(\alpha\sigma) \approx \frac{1}{2}\exp(\alpha\sigma)$，主方程退化为：

$$\dot{\varepsilon} = A_2 \exp(\beta \sigma) \exp\left(-\frac{Q}{RT}\right)$$
两边取对数并假定温度 $T$ 不变，对 $\sigma$ 求偏导：

$$\beta = \left( \frac{\partial \ln\dot{\varepsilon}}{\partial \sigma} \right)_T$$
操作：以 $\sigma$ 为横坐标，$\ln\dot{\varepsilon}$ 为纵坐标作图，各温度下拟合直线的平均斜率即为 $\beta$。
3. 计算 $\alpha$
根据数学定义，过渡参数 $\alpha$ 为：

$$\alpha = \frac{\beta}{n_1}$$
第二步：求解主应力指数 $n$
有了 $\alpha$ 之后，我们回到主方程的对数形式：$\ln\dot{\varepsilon} = \ln A + n \ln[\sinh(\alpha \sigma)] - \frac{Q}{RT}$。
假定温度 $T$ 不变，对方程右侧的变量 $\ln[\sinh(\alpha \sigma)]$ 求偏导：

$$n = \left( \frac{\partial \ln\dot{\varepsilon}}{\partial \ln[\sinh(\alpha \sigma)]} \right)_T$$
操作：以 $\ln[\sinh(\alpha \sigma)]$ 为横坐标，$\ln\dot{\varepsilon}$ 为纵坐标作图。不同温度下拟合直线的平均斜率就是当前应变下的 $n$。$n$ 反映了材料变形时的黏性特征。
第三步：求解热变形激活能 $Q$
对主方程的对数形式重新整理，将包含 $T$ 的项孤立出来：

$$\ln[\sinh(\alpha \sigma)] = \frac{1}{n}\ln\dot{\varepsilon} - \frac{1}{n}\ln A + \frac{Q}{nRT}$$
此时，我们假定应变速率 $\dot{\varepsilon}$ 恒定，对自变量 $(1/T)$ 求偏导：

$$\left( \frac{\partial \ln[\sinh(\alpha \sigma)]}{\partial (1/T)} \right)_{\dot{\varepsilon}} = \frac{Q}{nR}$$
移项后得到 $Q$ 的严格求解公式：

$$Q = R \cdot n \cdot \left( \frac{\partial \ln[\sinh(\alpha \sigma)]}{\partial (1/T)} \right)_{\dot{\varepsilon}}$$
操作：以 $1/T$ 为横坐标，$\ln[\sinh(\alpha \sigma)]$ 为纵坐标，在相同应变速率下作图。求出各应变速率下拟合直线的斜率平均值（设为 $S$），则 $Q = R \times n \times S$。$Q$ 值的大小反映了TC4材料在此应变下发生塑性变形和动态再结晶的难易程度。
第四步：求解结构因子常数 $\ln A$
引入温度补偿的应变速率参数（Z参数）：

$$Z = \dot{\varepsilon} \exp\left(\frac{Q}{RT}\right) = A [\sinh(\alpha \sigma)]^n$$
两边取对数得到截距式方程：

$$\ln Z = \ln A + n \ln[\sinh(\alpha \sigma)]$$
操作：将当前应变下所有温度和速率计算出的 $\ln Z$ 作为纵坐标，$\ln[\sinh(\alpha \sigma)]$ 作为横坐标进行全局线性回归。该直线的截距即为 $\ln A$。
第三章：多项式拟合（建立参数与应变的联系）
完成第二章后，针对你切分出的每一个真应变点（假设有15个点），你都得到了一组独立的 $(\alpha, n, Q, \ln A)$。
由于热塑性变形是连续的力学过程，我们需要将这些离散的参数连续化。数学上，最高效的方法是使用高阶多项式进行回归拟合。
对于TC4材料，通常5次或6次多项式能够极好地捕捉动态硬化和软化的拐点。以参数 $\alpha$ 为例，其与真应变 $\varepsilon$ 的数学关系可表示为：

$$\alpha(\varepsilon) = B_0 + B_1\varepsilon + B_2\varepsilon^2 + B_3\varepsilon^3 + B_4\varepsilon^4 + B_5\varepsilon^5 + B_6\varepsilon^6$$
同理，对另外三个参数建立多项式方程：

$$n(\varepsilon) = C_0 + C_1\varepsilon + C_2\varepsilon^2 + C_3\varepsilon^3 + C_4\varepsilon^4 + C_5\varepsilon^5 + C_6\varepsilon^6$$

$$Q(\varepsilon) = D_0 + D_1\varepsilon + D_2\varepsilon^2 + D_3\varepsilon^3 + D_4\varepsilon^4 + D_5\varepsilon^5 + D_6\varepsilon^6$$

$$\ln A(\varepsilon) = E_0 + E_1\varepsilon + E_2\varepsilon^2 + E_3\varepsilon^3 + E_4\varepsilon^4 + E_5\varepsilon^5 + E_6\varepsilon^6$$
操作：利用最小二乘法（在Origin中选择 Polynomial Fit，或在Python中使用 numpy.polyfit），求出上述 $B, C, D, E$ 各组系数矩阵。至此，你的材料常数已经全部升级为关于真应变的连续函数。
第四章：公式代入与全曲线预测推导
现在，模型已经建立完毕。当给定任意工作条件（温度 $T$、应变速率 $\dot{\varepsilon}$、真应变 $\varepsilon$）时，如何反推出理论流变应力 $\sigma$？我们需要对主方程进行代数反解。
已知：

$$Z = A(\varepsilon) [\sinh(\alpha(\varepsilon) \sigma)]^{n(\varepsilon)}$$
将应力项剥离出来：

$$\sinh(\alpha(\varepsilon) \sigma) = \left( \frac{Z}{A(\varepsilon)} \right)^{\frac{1}{n(\varepsilon)}}$$
根据双曲正弦函数的定义 $\sinh(x) = \frac{e^x - e^{-x}}{2}$，其反函数（反双曲正弦）存在明确的对数表达形式：

$$\text{arsinh}(x) = \ln(x + \sqrt{x^2 + 1})$$
令 $x = \left( \frac{Z}{A(\varepsilon)} \right)^{\frac{1}{n(\varepsilon)}}$，代入反函数并移项，即可得到最终的流变应力预测解析解：

$$\sigma = \frac{1}{\alpha(\varepsilon)} \ln \left\{ \left(\frac{Z}{A(\varepsilon)}\right)^{\frac{1}{n(\varepsilon)}} + \sqrt{ \left(\frac{Z}{A(\varepsilon)}\right)^{\frac{2}{n(\varepsilon)}} + 1 } \right\}$$
其中，动态 Z 参数的计算式为：

$$Z = \dot{\varepsilon} \exp\left( \frac{Q(\varepsilon)}{RT} \right)$$
